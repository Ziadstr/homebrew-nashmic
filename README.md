# Homebrew Tap for NashmiC

[NashmiC](https://github.com/Ziadstr/nashmic) is the first Franco-Arab programming language. Easy like Python, powerful like Go, Jordanian to the bone.

## Install

```bash
brew tap Ziadstr/nashmic
brew install nashmic
```

To install the latest development version from `main`:

```bash
brew install --HEAD nashmic
```

## Usage

```bash
# Run a NashmiC file
mansaf run hello.nsh

# Build to binary
mansaf build hello.nsh

# REPL
mansaf repl

# Format code
mansaf fmt hello.nsh
```
